# Defrese-e-commerce

![Screenshot of Insomnia testing](./assets/Insomnia%20Screenshot.png)


## Purpose
To create the backend of an e-commerce site that allows the user to update, add, get and delete products, product tags, and categories.

## Built With:
- JavaScript
- Insomnia
- Expressjs
- MySQL
- Sequelize
